> Darbot Yumlog is a lightweight PowerShell-based screen capture and recording utility that acts as "vision" for AI by taking screenshots at set intervals and analyzing them.

Key Features:

  • High-performance screen recording using FFmpeg
  • Periodic screenshot capture at configurable intervals
  • Simple PowerShell CLI interface
  • No telemetry - 100% local execution
  • Auto-installs FFmpeg if not present
  • Configurable FPS, duration, and output locations

About Yumlog

Darbot Yumlog is a lightweight PowerShell-based screen capture and recording utility that acts as "vision" for AI by taking screenshots at set intervals and analyzing them.

Key Features:

  • High-performance screen recording using FFmpeg
  • Periodic screenshot capture at configurable intervals
  • Simple PowerShell CLI interface
  • No telemetry - 100% local execution
  • Auto-installs FFmpeg if not present
  • Configurable FPS, duration, and output locations
