üíª PowerShell Commands Reference

Essential commands for working with Yumlog:

# Start a screen recording
.\launchers\record.ps1 -Fps 30 -DurationSec 10 -OutFile .\myvideo.mp4

# Capture periodic screenshots
.\launchers\capture.ps1 -Fps 2 -DurationSec 5 -OutDir .\myscreens

# Using the unified CLI
.\launchers\yumlog.ps1 start -Fps 30 -DurationSec 15
.\launchers\yumlog.ps1 get
.\launchers\yumlog.ps1 count
.\launchers\yumlog.ps1 size
.\launchers\yumlog.ps1 config

# Install FFmpeg (if needed)
.\launchers\install.ps1

# Run tests
.\launchers\run-tests.ps1

‚ÑπÔ∏è About Yumlog

Darbot Yumlog is a lightweight PowerShell-based screen capture and recording utility that acts as "vision" for AI by taking screenshots at set intervals and analyzing them.

Key Features:

  • High-performance screen recording using FFmpeg
  • Periodic screenshot capture at configurable intervals
  • Simple PowerShell CLI interface
  • No telemetry - 100% local execution
  • Auto-installs FFmpeg if not present
  • Configurable FPS, duration, and output locations
